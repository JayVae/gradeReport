    Title: FeatFormulaErr2 (Formula Evaluation Shared Feature) common record part

    This record part specifies Formula Evaluation & Error Ignoring data for a sheet, stored as part of a Shared Feature. It can be found in records such as FeatRecord. For the full meanings of the flags, see pages 669 and 670 of the Excel binary file format documentation and/or https://msdn.microsoft.com/en-us/library/dd924991%28v=office.12%29.aspx
